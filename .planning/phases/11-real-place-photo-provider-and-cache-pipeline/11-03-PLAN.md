## Truths

- UI must communicate source/attribution enough that provider/app media is distinguishable from missing state.
- UI must never imply an AI/illustrative image is a real place photo.
- API docs must describe real-photo sources, missing-photo fallback, and server-only provider boundaries.
- Final verification must include secret/logging checks.

</must_haves>
