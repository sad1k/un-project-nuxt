   - Create `tests/server/admin-auth.test.mjs`.
   - Assert `lib/db/schema/auth.ts` defines a `role` field on the `user` table with default `"user"` or equivalent safe default.

   - Assert the admin helper returns or throws 403 for authenticated non-admin users.
   - Assert Phase 9 does not add role-management UI/routes such as admin user role editing.

   - Update `lib/db/schema/auth.ts` to add a user `role` column.
   - Keep role values narrow in TypeScript, at minimum `"user" | "admin"`.

   - Preserve Better Auth adapter compatibility.
